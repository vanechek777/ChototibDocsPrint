Шаблоны импорта данных — ГПОУ «ЧТОТиБ»
==========================================

Импорт в программе выполняется ОТДЕЛЬНО для каждого типа данных
(Настройки / Админ / Группы / Студенты или мастер подключения к 1С).

Порядок импорта в пустую базу:
  1. Специальности  →  1_Специальности.csv (или .xlsx)
  2. Группы         →  2_Группы.csv
  3. Предметы       →  3_Предметы.csv
  4. Студенты       →  4_Студенты.csv

Формат: CSV (UTF-8, разделитель «;») или Excel (.xlsx).
Первая строка — заголовок, при импорте пропускается.
Для Excel используется только первый лист книги.

Файл «Импорт_ЧТОТиБ.xlsx» содержит все четыре листа для удобного
редактирования. Для импорта сохраните нужный лист в отдельный .xlsx
или используйте готовые CSV в этой папке.

Студенты: колонка «ID группы» — числовой Id из списка групп в программе
(после импорта групп откройте раздел «Группы» и уточните Id).

Пересоздать Excel из CSV:
  dotnet run --project Tools\GenerateImportTemplates
